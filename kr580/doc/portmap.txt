
    F0  READ    Принятые данные SPI
        WRITE   Данные на отправку
------------------------------------------------------------------------
    F1  READ    Статус
                - Bit 0: Готовность =1
                - Bit 1: Таймаут

        WRITE   Команда
                - 0 Запуск 80 тактов в SLOW режиме (инициализация)
                - 1 Чтение-запись в SPI
                - 2 CS=0 Отключение чипа
                - 3 CS=1 Включение чипа
------------------------------------------------------------------------
    F2  WRITE   Бит 0: при переключении с 0 на 1 - запуск команды
                Не забыть переключить обратно с 1 на 0
------------------------------------------------------------------------
    FE  READ    Номер клавиши
        WRITE   Bit 2-0: Цвет бордера
------------------------------------------------------------------------
    FF  READ    Счетчик нажатых клавиш
